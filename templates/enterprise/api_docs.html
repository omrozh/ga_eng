GrandAssembly

GAIA API Dokümantasyonuna Hoşgeldiniz.

Bu dokümantasyon ile GAIA ile nasıl özel karaktere sahip bir Yapay Zeka modeli oluşturup kullanabileceğinizi tanıtacağız.

GAIA Karakter Modeli Oluşturma

/api/train

POST

Buraya yollayacağınız bir request ile GAIA bir GAIA modeli oluşturabilirsiniz.

Request Formatı

{
 "character_description": "type:string",
 "api_key": "type:string"
}

Yanıt Formatı

{
 "character_id": "type:string",
}

GAIA Oluşturulan Karakter Modeline Prompt Yazdırma

/api/generate

POST

Buraya yollayacağınız bir request ile oluşturduğunuz modele prompt yazdırabilirsiniz.

Request Formatı

{
 "character_id": "type:string",
 "api_key": "type:string"
 "prompt": "type:string",
}

Yanıt Formatı

{
 "response": "type:string",
 "token_cost": "type:integer",
}
